# Our Calendar — P & B v5

The ultimate couples calendar app. Real-time sync, AI-powered recommendations, and everything you need to make the most of London together.

## Features
💗💙 Dual themes · 📅 Shared calendar · 🔄 Recurring events · ☀️ 7-day weather
🎂 Birthday & anniversary countdowns · 😊 Mood/vibe tagging · 🍽️ Restaurant rankings
💕 AI date planner · ✈️ Guest itinerary generator · 📰 Monthly AI wrap-ups
🎯 Couples goals dashboard · 🧠 AI concierge · 📲 Push notifications
📍 Map of places visited · 📸 Photo gallery · 📝 Wishlist · 🔗 Auto booking links
🔥 Live London events (refreshed weekly by Claude)

## Setup
1. Edit `src/firebase.js` with your Firebase config
2. Deploy to Vercel
3. Add `ANTHROPIC_API_KEY` + `FIREBASE_DATABASE_URL` as env vars
4. Add to both iPhone home screens
